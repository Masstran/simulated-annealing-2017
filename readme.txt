Данные программы собираются при помощи Visual Studio Community 2017 в операционной системе Windows 10 x64. 

Запуск генератора тестов:
task4_ProblemGenerator.exe
Генератор тестов работает без параметров командной строки и генерирует тесты на основе параметров, заданных константой в коде. 

Запуск программы, реализующей алгоритм возможен несколькими способами:

1)task4.exe
При запуске программы без параметров, будет выведена подсказка по запуску этой программы.

2)task4.exe <PathToData.xml> [-l]
PathToData.xml - путь к файлу с входными данными, оформленными в соответствии с требуемой структурой. 
При запуске без аргумента -l, программа выводит одно число: время работы расписания, полученного в результате работы алгоритма.
При запуске с аргументом -l, программа дополнительно выводит число итераций, через которое прошёл алгоритм, а также само оптимальное расписание.

3)task4.exe -p <M> <N> <a1> ... <aN> [-l]
При запуске с параметром -p появляется возможность ввода задания вручную. M - натуральное число - число процессоров. N - натуральное число - число работ. ai - натуральное число - время выполнения i-й работы.
При запуске без аргумента -l, программа выводит одно число: время работы расписания, полученного в результате работы алгоритма.
При запуске с аргументом -l, программа дополнительно выводит число итераций, через которое прошёл алгоритм, а также само оптимальное расписание.
